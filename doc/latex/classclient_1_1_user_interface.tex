\subsection{\-Detailed \-Description}
\-This is the user interface for \-Social \-Wombat.

\-This class is for anything dealing with what the user sees and the work flow of the social network. \-However, it does not deal with the behind-\/the-\/scenes logic for any of the actions, which is done in the client class.
