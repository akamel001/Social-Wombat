boolean client.Client.changePassword ( char[]  oldPassword,
char[]  newPassword,
char[]  confirmNewPassword,
String  userNameTemp,
String  currentUserName 
)

This a method for changing a user's password. Before changing a password, it must: verify that userNameTemp matches currentUserName; verify that oldPassword matches currentUserName's password; check that newPassword1 matches newPassword2. This method deals with passwords and care must be taken to leave no trace of any passwords, new or old.

Parameters:
oldPassword<- this is currentUserName's initial password
newPassword<- this is the desired password for currentUserName
confirmNewPassword<- again, this is the desired password for currentUserName (we ask for it again to minimize the effects of user typos)
userNameTemp<- this is the supplied name of the current user; it must match currentUserName for a password change
currentUserName<- this is the name of the current user
Returns:
